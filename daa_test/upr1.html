Упражнение 1: Асимптотични нотации


Def: Ще казваме, че функцията \(f: \mathbb{R}^+ \rightarrow \mathbb{R}\) е асимптотично неотрицателна, ако: $$\exists n_0 \in \mathbb{N} \enspace \forall n \geq n_0 : f(n) \geq 0$$             Ако неравенството е строго, ще казваме, че \(f\) е асимптотично положителна.

Def: Въвеждаме следните класове от функции относно асимтотично неотрицателната \(f\). Ще ги наричаме асимптотични нотации: $$O(f) = \{ g \, | \, \exists c > 0 \enspace \exists n_0 \in \mathbb{N} \enspace \forall n \geq n_0 : 0 \leq g(n) \leq c.f(n)\}$$             Неформално \(O(f)\) представлява класът на всички функции, които растят асимптотично не по-бързо от \(f\). Ще отбелязваме това чрез \(g \in O(f)\), \(g = O(f)\) или \(g \preceq f\). $$o(f) = \{ g \, | \, \forall c > 0 \enspace \exists n_0 \in \mathbb{N} \enspace \forall n \geq n_0 : 0 \leq g(n) \lt c.f(n)\}$$             Неформално \(o(f)\) представлява класът на всички функции, които растят асимптотично по-бавно от \(f\). Ще отбелязваме това чрез \(g \in o(f)\), \(g = o(f)\) или \(g \prec f\). $$\Omega(f) = \{ g \, | \, \exists c > 0 \enspace \exists n_0 \in \mathbb{N} \enspace \forall n \geq n_0 : 0 \leq c.f(n) \leq g(n)\}$$             Неформално \(\Omega(f)\) представлява класът на всички функции, които растят асимптотично не по-бавно от \(f\). Ще отбелязваме това чрез \(g \in \Omega(f)\), \(g = \Omega(f)\) или \(g \succeq f\). $$\omega(f) = \{ g \, | \, \forall c > 0 \enspace \exists n_0 \in \mathbb{N} \enspace \forall n \geq n_0 : 0 \leq c.f(n) \lt g(n)\}$$             Неформално \(\omega(f)\) представлява класът на всички функции, които растят асимптотично по-бързо от \(f\). Ще отбелязваме това чрез \(g \in \omega(f)\), \(g = \omega(f)\) или \(g \succ f\). $$\Theta(f) = \{ g \, | \, \exists c_1 \gt 0, \, c_2 \gt 0 \enspace \exists n_0 \in \mathbb{N} \enspace \forall n \geq n_0 : 0 \leq c_1.f(n) \leq g(n) \leq c_2.f(n)\}$$             Неформално \(\Theta(f)\) представлява класът на всички функции, които растят асимптотично еднакво с \(f\). Ще отбелязваме това чрез \(g \in \Theta(f)\), \(g = \Theta(f)\) или \(g \asymp f\).
